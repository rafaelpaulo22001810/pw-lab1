HTML5 e CSS

HTML5

HTML5 (Hypertext Markup Language, versão 5) é uma linguagem de marcação para a World Wide Web

Elementos Semânticos Foi usado? Descrição
Header 👍 Indica cabeçalho, todas as páginas contêm
Nav 👍 Contém o menu de navegação do website, está em todas as páginas
Main 👍 Foi utilizado neste ultimo laboratorio, server para colocar o código que nao se repete
Article 👎 Parte do decumento que pode ser vista de forma independente
Section 👍 Para separar conteudos, foi usada este mesmo pedaço de código
Aside 👎 Serve para fazer algumas anotações e publicidades
Footer 👍 Serve para fazer de rodapé, para não fica no fim da página automáticamente

CSS

CSS é chamado de linguagem Cascading Style Sheet e é usado para estilizar elementos escritos em uma linguagem de marcação como HTML.

Tipo de seletor Foi usado? Descrição
Universal 👎 Utilizado para aplicar o estilo a todos os elementos da página
Tipo 👍 Altera o estilo de um tipo de elemento, por exemplo alterar todos os elementos do tipo h3
Classe 👍 Cria uma classe com algum tipo de alteração e podemos escolher onde usar
Id 👎 Serve para identificar um elemento especifico, foi utilizado no quizz para identifcar as perguntas com multiplas respostas
Atributo 👎 Utilizado para aplicar algo a um certo atributo de um elemento
Pseudo-Classe 👎 Associado ao estado de um elemento
Pseudo-Elementos 👎 Regras que se aplicam a partes de elementos

Combinadores de seletores Foi usado? Descrição Simbolo
Descendentes 👎 Qualquer alteração é aplicada a tudo o que está dentro do div (espaço)
Filhos 👎 Aplicado ao que está diretamento dentro do div >
Adjacentes 👎 Aplicado a todos os P ques estao depois de um Div ~
Imediatamente Adjacente 👎 Aplicado ao P que está imediamente depois de um div +
